Project - Enhancing credit card fraud detection with SMOTE

card_transdata.csv is our dataset. Make sure the csv file is in the same path as the code file.

code_file.ipynb is the main code file. To run this file, you will need the following libraries -
1) pandas
2) seaborn
3) matplotlib
4) sklearn
5) collections
6) imblearn

UI_code.py is our user interface file. We can run this file directly from ubuntu terminal with command -
python UI_code.py
